# weather-app

🌦️ Weather App

A simple and modern Weather Application built with HTML, CSS, and JavaScript that fetches real-time weather data using the WeatherAPI
.

✨ Features

🌍 Search weather by city name

🌡️ Displays temperature, feels like, humidity, wind speed, and condition

🎥 Dynamic video backgrounds (Sunny, Rainy, Cloudy, Snowy, Mist, Thunder) based on weather

🏷️ Air Quality Index (AQI) with color-coded badge

📍 Shows local time and location

📱 Responsive UI with modern glassmorphism design

🖼️ Preview



🚀 Technologies Used

HTML5 for structure

CSS3 (Flexbox, Grid, Glassmorphism) for styling

JavaScript (ES6+, Async/Await, Fetch API) for logic

WeatherAPI for live weather & AQI data

⚡ How to Run Locally

Clone this repo

git clone https://github.com/RaviNani024/weather-app.git


Open the project folder

cd weather-app


Open index.html in your browser
